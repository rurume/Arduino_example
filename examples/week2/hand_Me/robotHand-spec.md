# robotHand-spec

## Servo Motor (輝盛 TOWERPRO MG995):四個

+ 0.5ms(0度)~2.2ms(180度)
+ Rotational Range:180°(另外一種版本是360°連續旋轉)
+ Pulse Cycle:20 ms
+ Pulse Width:1000-2000 µs
+ 轉力矩: 
	- 0.20sec/60degree(4.8v)
	- 0.16sec/60degree(6v) 
+ Stall torque:
	- 8.5kg/cm(4.8V)
	- 10kg/cm(6V) 
+ 工作電壓:4.8V-6V
+ Dead band width:5us 
+ 腳位說明   
	- 棕色：GND 
	- 紅色：4.8V~6V 
	- 橙色：PWM 輸入
